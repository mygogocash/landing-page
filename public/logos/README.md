# Official Logos

## GoGoCash Logo

Place your official GoGoCash logo files in this directory:

- **File name**: `gogocash-logo.svg` (preferred) or `gogocash-logo.png`
- **Recommended dimensions**: 
  - Header: 140x45px (or maintain aspect ratio)
  - Footer: 120x40px (or maintain aspect ratio)
- **Format**: SVG (preferred) or PNG with transparent background
- **Color**: Should work on both light and dark backgrounds

The logo component will automatically:
- Fall back to text "GoGoCash" if the image is not found
- Scale appropriately for different screen sizes
- Maintain aspect ratio

## Adding Your Logo

1. Place your official logo file as `gogocash-logo.svg` or `gogocash-logo.png` in this directory
2. The logo will automatically appear in the header and footer
3. If the file is not found, the component will display "GoGoCash" as text
