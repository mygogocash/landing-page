# Social Media Logos

## Adding Official Social Media Logos

Place your official social media logo files in this directory.

### File Naming Convention

- Use the platform name as the filename
- Format: `{platform}.svg` or `{platform}.png`
- Examples:
  - `facebook.svg`
  - `line.svg`
  - `instagram.svg`
  - `twitter.svg`
  - `youtube.svg`

### Recommended Specifications

- **Format**: SVG (preferred) or PNG with transparent background
- **Size**: 20x20px to 24x24px
- **Aspect Ratio**: Square (1:1)
- **Background**: Transparent
- **Color**: Official brand colors

### Supported Platforms

- `facebook` - Facebook logo
- `line` - Line Official Account logo
- `instagram` - Instagram logo
- `twitter` - Twitter/X logo
- `youtube` - YouTube logo

### Fallback Behavior

If a social media logo is not found:
- The component will display a Lucide React icon as fallback
- The button will function normally
